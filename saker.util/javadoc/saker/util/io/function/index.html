package saker.util.io.function
Package containing functional interfaces which can be used in an I/O error prone context.
Interfaces
IOBiConsumer
Functional interface similar to BiConsumer but is usable in an I/O error-prone context.
IOBiFunction
Functional interface similar to BiFunction but is usable in an I/O error-prone context.
IOConsumer
Functional interface similar to Consumer but is usable in an I/O error-prone context.
IOFunction
Functional interface similar to Function but is usable in an I/O error-prone context.
IORunnable
Functional interface similar to Runnable but is usable in an I/O error-prone context.
IOSupplier
Represents a supplier of results which may throw an IOException.
IOTriConsumer
Functional interface similar to TriConsumer but is usable in an I/O error-prone context.
IOTriFunction
Functional interface similar to TriFunction but is usable in an I/O error-prone context.
ObjectReaderFunction
Functional interface for reading objects from an arbitrary object input.
ObjectWriterFunction
Functional interface for writing objects to an arbitrary object output.

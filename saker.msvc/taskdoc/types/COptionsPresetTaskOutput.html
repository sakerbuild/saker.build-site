COptionsPresetTaskOutput
In bundle: saker.msvc
Contains one or multiple C option presets. C option presets are pre-made C configurations to be used with compiler and linker tasks to for given use-case. Presets can be acquired by using the saker.msvc.coptions.preset() task. The object can be directly passed to the saker.msvc.ccompile() and saker.msvc.clink() compiler options parameters.

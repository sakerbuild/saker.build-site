saker.msvc.sdk.windowskits()
SDKDescription
In bundle: saker.msvc
Gets an SDK description for the Windows Kits that matches the given versions. This task will create an SDK description that references the Windows operating system SDK with any of the specified versions. The returned SDK can be specified as an input for the SDKs parameter to the saker.msvc.ccompile() and saker.msvc.clink() tasks.

Version
Specifies the suitable Windows Kits versions that can be used for this SDK. The version numbers are expected to have the same format as they are under the Windows Kits\<OS>\bin directory if they are installed in the Program Files of the system. E.g.: 10.0.18362.0

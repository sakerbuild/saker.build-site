public static String getWindowsUapApiContractWinmdSDKPathIdentifier(String contractname, String version) throws NullPointerException
Gets an SDK path identifier for the Windows UAP SDK that locates an API contract .winmd file for the specified contract name.

The path of an API contract .winmd is usually the following: 

 Windows Kits/10/References/<SDK_VERSION>/<CONTRACT_NAME>/<CONTRACT_VERSION>/<CONTRACT_NAME>.winmd

E.g.: 

 Windows Kits/10/References/10.0.18362.0/Windows.Foundation.FoundationContract/3.0.0.0/Windows.Foundation.FoundationContract.winmd

The <SDK_VERSION> is the version of the SDK the path is retrieved from, not part of the path identifier.

contractnameThe contract name.
versionThe contract version.
The path identifier.
NullPointerExceptionIf any of the arguments are null.
public static SDKPathCollectionReference getWindowsUapApiContractsWinmdSDKPathCollectionReference()
Gets an SDK path collection reference that returns the .winmd files for the default API contracts for the platform.

The path collection reference will collect the paths for the .winmd API contracts defined in the Platform.xml file for the UAP SDK.

The SDK path collection reference.

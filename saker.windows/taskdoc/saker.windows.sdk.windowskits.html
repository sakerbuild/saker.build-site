TaskDoc Index
saker.windows.sdk.windowskits()
SDKDescription
In bundle: saker.windows
Gets an SDK description for the Windows Kits that matches the given versions. This task will create an SDK description that references the Windows operating system SDK with any of the specified versions.
Summary
Parameters
VersionsSpecifies the suitable Windows Kits versions that can be used for this SDK.
Parameters
Versions
Collection of String
Aliases:
<unnamed>
Version
Specifies the suitable Windows Kits versions that can be used for this SDK. The version numbers are expected to have the same format as they are under the Windows Kits\<OS>\bin directory if they are installed in the Program Files of the system. E.g.: 10.0.18362.0

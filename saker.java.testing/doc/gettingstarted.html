Getting started

The saker.java.testing package runs as part of the saker.build system. The packages are distributed using the saker.nest repository. In order to install them, see the installation guides for saker.build and saker.nest.

You don't need anything else to set up, you can use the saker.java.testing package by referencing the contained tasks in it.

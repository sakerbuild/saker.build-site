Release 0.8.0 (initial)

2020 January 6

Version 0.8.0 is the initial release of the saker.build system. It's out, and you can get started using it inside an IDE or on the command line. Visit the installation guide for more information.

If you are curious why 0.8.0 was chosen for the initial release number, read the 0.8.0 seems like an odd initial release number blog post.

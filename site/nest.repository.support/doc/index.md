# Overview

The [nest.repository.support](https://nest.saker.build/package/nest.repository.support) package in the [saker.nest repository](root:/saker.nest/index.html) provides features for accessing the saker.nest repository features using the [saker.build system](root:/saker.build/index.html).

The build tasks in this package can be used to manage bundles for the given bundle storages of the repository, and to work with dependencies.

The package also provides a programmatic API for other package authors to integrate with various aspects of the Nest repository.

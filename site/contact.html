Contact

The contact points for getting in touch with the saker.build maintainers and developers are categorized based on the topic.

Software bugs

If you encounter any bugs or issues with the saker.build system and related software, we recommend you to file an issue in the appropriate GitHub repository:

  • Saker.build system: sakerbuild/saker.build
  • Saker.nest repository: sakerbuild/saker.nest
  • For other packages and plugins see the associated source code repository.

Saker.nest repository problems

If you encounter any problems working with the saker.nest repository you can write a direct e-mail to: support@saker.build

The problems that you e-mail us with should be related to your account only. If you encounter problems that can be publicly displayed, you should file an issue at the sakerbuild/saker.nest GitHub repository instead. Such personal issues may include:

  • You can't create an account.
  • You don't receive the e-mails we send to you (not even in the spam folder).
  • Your bundle upload API keys don't work even after regenerating them.
  • You can't verify your account.
  • Your published package doesn't show up.
  • You want us to enable a restricted package or task name for you.
  • You want us to increase the limits for your account.

Tech support

If you have a question related to saker.build and the related ecosystem, please refer to the Documentation. If you still encounter issues, we recommend asking a question on Stack Overflow.

If you'd like to receive paid support please write an e-mail to: support@saker.build

Note that we will not be providing direct individual tech support through the above e-mail.

Individal issues

If you have an issue that contains sensitive data, or specifically related to you, write an e-mail to: support@saker.build

Licensing

If you wish to use software maintained by the saker.build team under a different license, please write an e-mail to: support@saker.build

General

In other cases that doesn't fit any of the above topics, you can write an e-mail to: support@saker.build

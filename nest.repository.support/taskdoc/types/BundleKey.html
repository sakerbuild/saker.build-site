BundleKey
In bundle: nest.repository.support
An unique identifier object for a saker.nest bundle for the current storage configuration. The object uniquely identifies a bundle that is available in the current repository storage configuration. It uniquely identifies a bundle even if a bundle identifier is present multiple times in the configuration. This object can be passed to tasks that accept bundle keys as inputs.

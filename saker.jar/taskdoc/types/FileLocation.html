FileLocation
In bundle: saker.standard
Related types:
LocalFileLocation
ExecutionFileLocation
Represents a location of a given file. A file location is a reference to a file that exists at some place in the build environment. It may be either an execution file location, or a local file system location. The path of the file may be retrieved by using the field [Path] for execution and [LocalPath] for local file locations.

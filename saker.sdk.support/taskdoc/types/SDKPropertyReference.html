SDKPropertyReference
In bundle: saker.sdk.support
Reference to property in a given SDK. Represents a property in a SDK that is dynamically resolved based on the SDK context of where the object is applied. Using this only makes sense in a context where there are SDK definitions, against which this property reference can be resolved.

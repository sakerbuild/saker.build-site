sdk.property()
SDKPropertyReference
In bundle: saker.sdk.support
Creates a reference to a property in a given SDK. An SDK property reference constists of an SDK Name from which it looks up the property. A property Identifier that specifies the property that is contained in the SDK. SDK property references can be passed to tasks which support it. They are resolved against the SDKs which are also passed to the associated task as input. SDK property references are not useful on their own and they should always be used with other SDK inputs. They can be used to specify inputs which are not directly part of the build execution but are installed in the supported build environment.

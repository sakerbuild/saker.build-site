package saker.sdk.support.api
Contains classes for SDK representation and management.
Interfaces
EnvironmentSDKDescription
SDKDescription that represents an SDK that is resolved from a given build environment.
IndeterminateSDKDescription
SDKDescription that may possibly select different SDKs on different environments.
ResolvedSDKDescription
SDKDescription that provides access to an already resolved SDKReference.
SDKDescription
Represents a description for an SDK.
SDKDescriptionVisitor
Visitor interface for the possible types of SDKDescription.
SDKPathReference
Interface that encloses a reference to an SDK path.
SDKPropertyReference
Interface that encloses a reference to an SDK property.
SDKReference
Provides access to the paths and properties of an SDK.
UserSDKDescription
SDKDescription that encloses a predefined set of paths and properties.
Classes
SDKSupportUtils
Utility class providing functions for SDK management.

package saker.build.launching
public final class Main
saker.build.launching.Main
java.lang.Object
Main entry point to the build system for command line usage.

This class contains the main(String...) function for the build system. This can be called directly from the command line, or in other ways if needed to be called programmatically.

The build system doesn't ever call System.exit(int) by itself, so it is not expected to terminate the JVM. (However, malicious third party plugins may interfere with this, but this should not be considered under normal use-case.)

The main method will either finish successfully, or throw an exception if failed.

The arguments will be parsed based on the command line interface of the build system, and the appropriate actions will be taken.

package saker.std.api.environment.qualifier
public interface EnvironmentQualifierVisitor
Visitor interface for possible types of EnvironmentQualifier.

The visitor is used with EnvironmentQualifier.accept(EnvironmentQualifierVisitor) where the subject environment qualifier will call the appropriate visit method of this interface.

All of the declared methods in this interface are default and throw an UnsupportedOperationException by default. Additional visit methods may be added to this interface with similar default implementations.

Clients are recommended to implement this interface.

std.file.put()
FileLocation
In bundle: saker.standard
Puts a file to the given file location. The task takes the string Contents and writes them to the specified file location. It can be used to write a file directly from the build script. Developers should take great care when using this task, as using the written file by other tasks can lead to concurrency errors and inconsistent builds. Generally, this task should be used to write a file as an output of the build and shouldn't be consumed by other tasks without additional task synchronization.
